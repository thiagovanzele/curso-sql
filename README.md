# Curso de SQL com PostgreSQL

Repositório criado para armazenar exercícios e scripts
desenvolvidos durante o curso de SQL utilizando PostgreSQL
e pgAdmin.

## Conteúdo
- Consultas básicas (SELECT, WHERE, ORDER BY)
- JOINs
- Subqueries
- Funções agregadas
- Criação de tabelas

## Ferramentas
- PostgreSQL
- pgAdmin
